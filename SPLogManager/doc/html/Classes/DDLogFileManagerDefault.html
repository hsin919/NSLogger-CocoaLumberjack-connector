Overview

Default log file manager.

All log files are placed inside the logsDirectory. If a specific logsDirectory isn’t specified, the default directory is used. On Mac, this is in ~/Library/Logs/. On iPhone, this is in ~/Library/Caches/Logs.

Log files are named “ .log” Example: com.organization.myapp 2013-12-03 17-14.log

Archived log files are automatically deleted according to the maximumNumberOfLogFiles property.
